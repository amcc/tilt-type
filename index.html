Variable font - mobile device tilter

Please tap to begin, then tilt and spin. This page uses a mobile phone's tilt and compass hardware to vary aspects of the type.

  • tilt the phone up and down to control weight
  • left and right to control how casual the type is
  • rotate (or turn around) for slant

Google's variable font Recursive allows for various settings to be varied, you can see more about that on their variable font page
